## 📌 Project Description

This application is a basic text editor that provides a menu-driven interface similar to professional IDEs.  
It was developed in Eclipse IDE as part of Java GUI practice.

---

## 🚀 Features

- 📝 Text editing using JTextArea  
- 📂 File menu (Open, Save, Exit)  
- ✂️ Edit menu (Copy, Cut, Redo)  
- 🔍 Search menu options  
- 🛠 Source and Refactor options  
- ▶ Run and Debug menu  
- 🪟 Window and Help menu
